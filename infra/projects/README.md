# Projetos

Cada subpasta aqui e um template operacional para um projeto independente no Coolify.

Use como base:

- `.env.example` para variaveis esperadas;
- `docker-compose.prod.yml` quando o projeto precisar de mais de um processo;
- `README.md` com notas especificas de deploy.

Projetos sem compose ainda devem ser criados no Coolify como app Dockerfile/Nixpacks ou receber um compose proprio antes do deploy.
