# Terminology

| Term | Definition |
|---|---|
| Product | Aggregated product from multiple marketplaces |
| Marketplace | Integrated e-commerce platform |
| Offer | Marketplace-specific product offer |
| PriceHistory | Historical price snapshot |
| Alert | User-defined price monitoring rule |
| Crawler | Service responsible for external data collection |
| Worker | Async process responsible for jobs |
| Queue | Async processing queue |
| AI Engine | Recommendation and analysis system |
| Scheduler | Recurring task execution service |
| Normalization | Data standardization process |
| Premium User | User with advanced features |
