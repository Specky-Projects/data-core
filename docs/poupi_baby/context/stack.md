# Recommended Stack
| Layer | Technology | Reason |
|---|---|---|
| Frontend | Next.js | SEO, SSR and performance |
| Frontend Language | TypeScript | Type safety and maintainability |
| Styling | TailwindCSS | Fast and consistent UI development |
| Backend | NestJS | Modular and scalable architecture |
| Backend Language | TypeScript | Fullstack standardization |
| Database | PostgreSQL | Consistency and reliability |
| Cache | Redis | Performance and caching |
| Queue System | BullMQ | Async processing for MVP |
| Distributed Messaging | RabbitMQ (Future) | Distributed event processing |
| Infrastructure | Docker Compose | Simple MVP infrastructure |
| Future Scaling | Kubernetes | Horizontal scaling |
| Monitoring | Sentry | Error tracking |
| CDN & Security | Cloudflare | Global performance and protection |
| Frontend Hosting | Vercel | Optimized for Next.js |
| Backend Hosting | Railway / Render | Simplified backend deployment |
