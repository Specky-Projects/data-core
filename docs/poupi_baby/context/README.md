# Poupi Context

Este diretório contém toda a documentação estratégica, técnica e arquitetural do projeto Poupi.

Os documentos aqui servem como:
- base para desenvolvimento
- documentação IA-friendly
- contexto para code generation
- referência arquitetural
- documentação de produto

---

# Objetivos

- Centralizar contexto do projeto
- Facilitar onboarding
- Padronizar arquitetura
- Melhorar integração com IA
- Facilitar escalabilidade futura

---

# Filosofia do Projeto

Poupi deve priorizar:

- simplicidade
- velocidade de desenvolvimento
- experiência do usuário
- escalabilidade gradual
- automação
- inteligência de dados
- validação rápida
